# 학교 로고 이미지 안내

이 폴더(`assets`)에 사용자가 직접 제작한 3가지 학교 로고 이미지 파일을 아래의 파일명과 확장자로 저장해 주세요.

1. **tsukinowa.png** (츠키노와 학교 로고)
2. **sekisetsu.png** (세키세츠 학교 로고)
3. **katen.png** (카텐 학교 로고)

이미지를 넣은 후 웹페이지를 새로고침(또는 서버 재시작)하여 학교 테마 버튼을 클릭하면, 저장한 이미지가 학생증 카드 왼쪽 하단에 연동되어 정상적으로 나타납니다.
만약 파일이 준비되지 않은 상태에서 테마를 선택하면, 로고가 깨지지 않고 "로고 이미지 없음" 이라는 대체 안내 텍스트가 표시되도록 안전하게 설계되었습니다.
